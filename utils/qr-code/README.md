# About this script
This script generates a QR code from a GitHub URL.

# Run it
Use `python3 generate_code.py FULL_GITHUB_URL`.

E.g. 

`python3 generate_code.py https://github.com/ford-at-aws/2024-data-summit`

### Output
Should print
`QR code generated and saved as github_repo_qr.png`.

# Note
Only tested on MacOS Sonoma.
